# FamilyGo – V9.8.1

## Clean Documents Module

מודול המסמכים נבנה מחדש מחוץ ל-MainActivity:

- `DocumentRepository.kt` – קבצים, מצלמה, Offline ופתיחה.
- `DocumentsViewModel.kt` – מצב המסך ותהליך יצירת מסמך.
- `DocumentsModuleScreen.kt` – מסך Compose עצמאי.

### יכולות
- העלאת PDF ותמונות.
- צילום מסמך.
- עותק מקומי לגישה אופליין.
- חיפוש וסינון.
- מספר הזמנה, תאריך ושעה.
- קישור לטיסה, מלון או פעילות.
- FileProvider מאובטח.
- ללא OCR לא יציב.
